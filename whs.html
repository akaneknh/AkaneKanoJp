Student Housing Service

This is a fictitious Student Housing Service, built as a SPA (single page application) using React js, and using PHP/MySQL on back end. We made this website as a project for a Advenced JavaScript course at college.

Functional requirement

  • Sign Up and Sign in
  • Post Comment
  • Search Post
  • Display badges according to user status

Time frame

2 weeks (09/2022)

  • Henry - Developer

My Role

  • Project Management
  • Determination of functional requirements
  • Wireframe/Mock-up/Prototype
  • Coding (HTML/CSS/React js)

Badge System

Badges are displayed on the user's profile screen to indicate whether the user is a trustworthy user or not, depending on the identification documents registered at the time of account registration.
