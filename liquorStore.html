Project Detail

JAM Liquor

Check it on website

JAM Liquor is a fictitious online liquor store built with Angular JS. We made this website as a project for a Advenced JavaScript course at college.

Functional requirement

  • Editable shopping cart
  • Total amount is displayed on the payment screen
  • Display information of logged-in users
  • Display an age verification modal for first-time visitors
  • Products can be searched by keyword.

Time frame

2 weeks (08/2022)

Team Member

  • Me - Project Manager, UI/UX Designer, Developer
  • Jose - Developer
  • Matheus - Developer
  • Aplil - Developer

My Role

  • Project Management
  • Paper Prototype and Wireframe / Mock-up
  • Coding (HTML/CSS)

Paper Prototype

After reviewing the functional requirements, I created paper prototypes and shared them with the members of the team to confirm the flow lines for each function.

Wireframe

Final Design

We made 4 category pages, and built it as SPA(Single Page Aplication) to ensure smooth page transitions.
